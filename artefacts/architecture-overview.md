# Общая диаграмма кейса

Диаграмма описывает кейс целиком: от выбора сегмента до оценки результата эксперимента.

```mermaid
flowchart TD
    A[Все родители на платформе] --> B{Ребёнок активен<br/>на предпериоде?}
    B -- Нет --> B1[Исключаем из эксперимента<br/>('отточники')]
    B -- Да --> C[Целевой сегмент эксперимента]

    C --> D[Случайный сплит A/B]
    D --> E[Контрольная группа - КГ<br/>старый ЛК / старая подписка]
    D --> F[Целевая группа - ЦГ<br/>новый ЛК / новая подписка]

    F --> F1[Импорт школьных оценок]
    F1 --> F2[Gap Analysis:<br/>сравнение школа vs платформа]
    F2 --> F3[Рекомендации заданий<br/>для закрепления пробелов]
    F2 --> F4[Рекомендательная система:<br/>курсы/статьи по сильным темам]
    F3 --> F5[Обновлённая плашка подписки]
    F4 --> F5

    E --> G[Сбор поведенческих событий]
    F5 --> G

    G --> H1[Метрика 1 - North Star:<br/>медианное время в ЛК за сессию]
    G --> H2[Метрика 2:<br/>кол-во уникальных вкладок ЛК за сессию]
    G --> H3[Метрика 3:<br/>время от старта сессии<br/>до перехода к оплате]

    H1 --> I1[Срез: есть подписка / нет подписки]
    H2 --> I1
    H3 --> I2[Срез: новая подписка vs старая подписка]

    I1 --> J[Сравнение ЦГ vs КГ]
    I2 --> J
    J --> K{Статистически значимый<br/>прирост вовлечённости<br/>и конверсии?}
    K -- Да --> L[Раскатка фичи на весь<br/>целевой сегмент]
    K -- Нет --> M[Доработка гипотезы /<br/>повторный тест]
```

## Как читать диаграмму

1. **Сегментация** — из всей базы родителей отбираются только те, чей ребёнок был активен на предпериоде (верхняя часть).
2. **Эксперимент** — сегмент случайно делится на ЦГ и КГ, чтобы группы были сопоставимы.
3. **Фича (только в ЦГ)** — импорт школьных оценок → анализ пробелов → два типа рекомендаций (закрепление пробелов и развитие сильных сторон) → новая подписка.
4. **Сбор данных** — события поведения родителя в ЛК логируются в обеих группах одинаково.
5. **Метрики** — считаются с нужными срезами (по наличию подписки для метрик 1–2, по версии подписки для метрики 3).
6. **Оценка** — сравнение групп и принятие решения о раскатке.
